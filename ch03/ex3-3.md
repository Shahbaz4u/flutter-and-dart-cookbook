# 3.3 Using a For statement

# Example: For loop

```dart
void main() {

  int maxIterations = 10;
  for (var i = 0; i < maxIterations; i++) {
    print ('Iteration: $i');
  }
}

```

# Example: For Each loop

```dart
void main() {
  List daysOfWeek = ['Sunday', 'Monday', 'Tuesday'];

  daysOfWeek.forEach((print));
}

```
