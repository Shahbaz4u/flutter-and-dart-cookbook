# 2.2 Using While/Do While


## Example: While loop

```dart
void main() {

  bool isTrue = true;

  while (isTrue) {
    print ('Hello');
    isTrue = false;
  }
}

```

## Example: Do While loop

```dart
void main() {

  bool isTrue = true;

  do {
    print ('Hello');
    isTrue = false;
  } while (isTrue) ;
}
```
