# 2.7 How to use Lists
 
## Example

```dart
void main() {
  var letters = ['a','b','c'];
  
  // Print the list
  print ('$letters');
  
  // Print an element from the list
  print ('${letters[0]}');
  
  // Print each element in the list
  letters.forEach((letter) => print ('Letter: $letter'));
}
```
