# How to use while/do while

## Example

#### Control Flow: while loop
```dart
void main() {

  bool isTrue = true;
  
  while (isTrue) {
    print ('Hello');
    isTrue = false;
  }
}
```

#### Control Flow: do while loop
```dart
void main() {

  bool isTrue = true;
  
  do {
    print ('Hello');
    isTrue = false;
  } while (isTrue) ;
}
```
