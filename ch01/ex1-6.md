# How to use while/do while

Control Flow: while loop
```
void main() {

  bool isTrue = true;
  
  while (isTrue) {
    print ('Hello');
    isTrue = false;
  }
}
```

Control Flow: do while loop
```
void main() {

  bool isTrue = true;
  
  do {
    print ('Hello');
    isTrue = false;
  } while (isTrue) ;
}
```
